# MachineLearning
It is a Brent crude oil price prediction project to avaliate and compare four Machine Learning and Deep Learning regression algorithms.

Brent crude oil prices strongly influence the global economy, yet their forecasting remains challenging due to market volatility and complex macroeconomic interactions. In this sense, this project evaluates machine learning methods for multivariate one-step-ahead Brent price forecasting using daily data from 2014–2023 and selected macroeconomic indicators.
The assessed approaches include Naive and ARIMA baselines, Ridge Regression (RR), Random Forest (RF), Support Vector Regression (SVR) and Long Short-Term Memory (LSTM). Models were validated on 2024 and tested on 2025, using MAE, MAPE and RMSE metrics. Among the learning-based models, RF achieved the best performance. However, neither RF nor the remaining machine and deep learning approaches outperformed the Naive and ARIMA baselines. Diebold–Mariano (DM) tests confirmed the competitiveness of persistence-based forecasts, indicating that simple statistical methods remain difficult to surpass for one-day-ahead Brent price prediction under the evaluated conditions.
